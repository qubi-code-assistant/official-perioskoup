## Technical SEO Requirements
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- Correct hreflang tags
- IndexNow integration
- Canonical URLs on every page
- XML sitemap at /sitemap.xml
- robots.txt allowing AI crawlers (GPTBot, ClaudeBot, PerplexityBot, etc.)
- Open Graph + Twitter Card meta on every page
- Structured data: SoftwareApplication, FAQPage, Organization, Physician

## GEO Readiness Requirements
- llms.txt deployed at /llms.txt
- llms-full.txt at /llms-full.txt
- All AI crawlers allowed in robots.txt
- FAQPage schema on every content page
- Answer capsules after every H2 (short summary paragraph for AI extraction)
- Dr. Anca's EFP quote prominent for citation
